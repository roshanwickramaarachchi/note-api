 # Note  API

 > Backend API for Note Application, which is a mobile app 

 ## Usage

 Rename "config/config.env.env" to "config/config.env" and update the values/setting to your own

 ## Install Dependencies

 ```
 npm install
 ```
 ## Run App

 ```
 # Run in dev mode
 npm run dev

 # Run in prod mode
 npm start
 ```
